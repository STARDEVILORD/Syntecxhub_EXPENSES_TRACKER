# Expense Tracker

A simple and responsive web application for managing daily expenses. Users can add, edit, and delete transactions while tracking their income, expenses, and available balance in real time.

## 🚀 Features

* Add and manage expense records
* Edit and delete transactions
* Track income and expenses
* Real-time balance calculation
* Responsive and user-friendly interface
* Local storage support for data persistence

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
Expense-Tracker/
├── index.html
├── style.css
├── script.js
└── assets/
```

## 🎯 Objective

The main objective of this project is to help users monitor their spending habits and manage personal finances efficiently through a simple web-based interface.
